# Dice.py
This Python program simulates the rolling of two dice, generating random numbers between 1 and 6 for each die to create a fun and interactive gaming experience.
It can be used for various purposes such as playing games like snake and ladder, ludo, business, etc. This code can be integrated with various new software /applications in which they want to rolll a die for their next step to come like lottery games, ludo in mobiles etc.
